4 Phases par axe

A · Places en temps réelQ3 2026 → Q1 2027
Impact × 10 pour l'usager. Le seul facteur vraiment différenciant.
  • A-0 Prototype (juillet 26) : 1 parking pilote (Cour de Gare ou Planta). Méthode : API exploitant → scraping → capteurs IoT. Statut disponible / presque plein / complet.
  • A-1 Généralisation (Q4 26) : 7 parkings via convention Ville. Chiffres exacts + graphique 24h historique + prédiction.
  • A-2 Temps réel complet (Q1 27) : WebSocket, notifs push "place libérée", historique 7/30/365 jours.
⚠️ Bloquant : accord formel avec au moins un exploitant ou la Ville. Fallback capteurs IoT LoRa (Swisscom) coûteux.
B · Parkings moto / 2-rouesJuillet 2026 · 2 semaines
Quick win à haute valeur : segment ignoré par tous les outils concurrents.
  • B.1 Recensement : relevé terrain ou demande Ville. ~15–20 zones (gratuites pour la plupart). Données : adresse, GPS, places, horodateur, tarif.
  • B.2 Intégration produit : type parking ∈ {voiture, moto, velo_securise}. Toggle UI "🚗 Voiture / 🏍 Moto". Simulateur adapté.
  • B.3 SEO : page dédiée /parkings-moto-sion. Requête cible : "parking moto Sion".
C · Bornes de recharge EVAoût 2026 · 3 semaines
  • C.1 Source de données : OpenChargeMap API (open data, CH couvert). Backup : ChargeMap / scraping evpass.ch.
  • C.2 Intégration : entité borne_ev (GPS, opérateur, kW, connecteurs, CHF/kWh). Carte avec filtre "⚡".
  • C.3 Tarification combinée : simulateur "me garer 3h ET recharger". Recommandation parking avec borne intégrée.
D · Commerces partenaires (Migros, Coop…)Septembre–Novembre 2026
L'axe le plus incertain : dépend de négociations commerciales.
  • D.1 Pré-démarchage (septembre) : Migros Métropole, Coop Sion, commerçants Grand-Pont. Pitch : "validation ticket parking".
  • D.2 MVP 1–2 partenaires (octobre) : page "Où valider mon ticket ?" Carte commerces proches de chaque parking.
  • D.3 Extension (novembre) : onboarding 10+ commerçants. Dashboard commerçant minimal.
  • D.4 Modèle économique (décembre) : gratuit / commission 0.10 CHF validation / forfait 150 CHF/an.
⚠️ Bloquant : accords commerciaux. Négociation Migros/Coop centralisée (3–6 mois). Alternative : commencer par commerçants indépendants.
E · Mobilité douce (vélos, trottinettes)Q1 2027 · 4 semaines
  • E.1 Vélos libre-service : API PubliBike (si Sion couvert) ou vélos communaux.
  • E.2 Trottinettes : Lime, Tier, Voi — à confirmer présence à Sion.
  • E.3 UX : carte unifiée 🚗 🏍 ⚡ 🚲 🛴. Recommandation "dernier km" : Potences (gratuit) + vélo pour centre.
F · P+R (Park & Ride)Q1 2027 · 6 semaines
  • F.1 Recensement P+R : CFF Sion Gare, P+R Uvrier, Brignon. Car Postal connexions régionales.
  • F.2 Intégration TP : API CFF Hafas. Itinéraire "voiture → gare → train → centre". Coût comparé.
  • F.3 Recommandation intelligente : "Samedi marché : P+R + bus = 8 CHF vs Planta = 15 CHF". Émissions CO2.
G · Intégration sion.chAvril 2026 → Q4 2026 · transversal
Porte d'entrée politique : sans ça, le reste reste confidentiel.
  • G.1 Prise de contact (avril–mai 26) : service mobilité + développement durable. RDV, alignement gouvernance.
  • G.2 Widget embarquable (juin–juillet 26) : <iframe src="...">. Version complète + mini-widget.
  • G.3 Sous-domaine officiel (septembre 26) : parking.sion.ch en CNAME. Co-branding.
  • G.4 Convention (Q4 26) : mission, durée, indicateurs, RGPD, continuité service.
